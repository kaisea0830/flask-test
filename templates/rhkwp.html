제작 주제: 개인 맞춤형 러닝 코스를 추천해주는 웹

1) Problem Statement Canvas

컨텍스트

운동을 하고 싶지만 거리나 시간에 맞는 러닝 코스를 찾기 어렵고, 날씨나 시간대에 따라 뛰기 적절한 환경을 알기 어려움

근본적인 문제

사용자 맞춤형 운동 코스 정보 부족, 런닝에 적합한 시간/날씨 정보 부재

고객

운동하고 싶은 20~40대 직장인, 러닝을 즐기는 MZ세대 건강 관심자

감정적 영향

  • 동기 부족
  • 귀찮음
  • 무기력감

정량적 영향

  • 운동 루틴 실패
  • 건강관리 실패
  • 좌식 생활 증가

대안

  • 유튜브 운동 루틴
  • 날씨 앱 확인
  • 메모장 코스 저장

대안의 단점

  • 정보 분산
  • 사용자 맞춤 아님
  • 코스 거리 예측 어려움

2) 솔루션 (3줄)

  • 사용자 위치 기반 + 거리/시간 입력을 통해 맞춤형 러닝 코스를 추천
  • 날씨, 기온, 시간대 데이터를 분석하여 런닝하기 좋은 시간대를 예측해 제공
  • 기록 저장 및 사용자 후기 반영으로 코스 품질을 지속적으로 개선

3) 기능 명세

  • 사용자 위치 기반 주변 러닝 코스 탐색
  • 러닝 거리(km) 또는 시간(min) 입력 기능
  • 지도 기반 경로 시각화
  • 날씨/기온/미세먼지 기반 추천 시간대 제공
  • 러닝 기록 저장 및 통계 제공
  • 사용자 선호 기반 개인화 추천

4) 간단한 UX 흐름

  • 홈 화면 → 내 위치 기반 자동 인식
  • → 거리 or 시간 설정
  • → 주변 추천 코스 출력 (지도 표시)
  • → 날씨 및 추천 시간대 정보 제공
  • → 러닝 시작 or 저장
  • → 러닝 완료 후 기록 저장 및 후기 작성
